# GW23 周报

这一周的关键词，大概可以浓缩成三句：

1. `D.Jenkins` 仍然是本周最忙的打工人。  
2. 扣分这玩意，平时看着像小伤，周末结算时往往直接变截肢。  
3. 有些对局看着差 1 分，实则是心脏骤停级别；有些看着只是小修小补，最后却打成了大屠杀。

下面进入本周战报。

## 尼弟 1129 vs 雕哥 1332

尼弟本周操作不算少，`DAY2` 先把 `I.Collier` 换成 `K.Knueppel`，`DAY4` 又把 `J.Holiday` 升级成 `D.Jenkins`，思路很清晰，就是奔着填坑去的。可惜到了后半周，局势一紧，尼弟还是没忍住在 `DAY6` 两连扣分，`Walker -> 文班`、`东契奇 -> Rollins`，直接 `-200`。  

雕哥这边则是典型的“稳着赢”。`DAY3` 先补 Jenkins，`DAY6` 再上 `J.Duren`，最后只吃了一个 `-100`。对比下来，尼弟是想追，雕哥是想收。  

203 分的分差属于那种，表面看是最后崩盘，实际上是整周都在交学费。尼弟本周不是没亮点，而是每次刚要抬头，就被扣分按了回去。雕哥则靠着更干净的操作完成了标准控场，拿下一场很有雕味的胜利。

## 堡 1423 vs 文史哲 1422

本周最刺激的一场，没有之一。  

堡这边一共就两笔操作，`DAY4 L.Shamet -> D.Jenkins`，`DAY7 L.Doncic -> T.Maxey`，属于“该出手时出手，不该出手时绝不乱动”。文史哲则完全是另一条路：`DAY2` 直接开 `WC`，一口气大修 6 人，后面又在 `DAY5` 把 `Brunson` 换回 `Doncic`，`DAY6` 再把 `Jokic` 卖成 `文班`。  

结果呢？文史哲这一通华丽施工，最后就差了 `1` 分。  

这就是范特西最残忍的地方。不是你没操作，也不是你操作得不够帅，而是你忙活了一整周，最后发现对面那种“只动两刀、刀刀见血”的人，才是真正的坏东西。堡 1 分险胜，这种赢法比大胜还伤人，因为输家赛后基本只会做一件事：对着自己的每一笔交易复盘到天亮。

## Paul 1395 vs 笨笨 1466

Paul 本周两手都挺像样：`DAY3 N.Traore -> D.Jenkins`，`DAY4 R.Westbrook -> K.Knueppel`，属于顺着热点走，也没什么逆天操作。  

问题在于，笨笨这周也没犯错。`DAY2 N.Clifford -> D.Carter`，`DAY7 M.Raynaud -> W.Riley`，两笔都够实用，整队从头到尾没什么明显短板。  

最后 71 分的分差，不算惨案，但也绝不是“运气差一点”。更像是 Paul 这周整体发挥正常，偏偏碰上了一个比自己再完整半档的对手。笨笨赢得不算夸张，却很稳，很像那种你觉得还有机会，抬头一看其实分差已经回不去了的对局。

## 小火龙 1348 vs 船哥 1371

小火龙本周最有个性的地方，是他一个人扮演了“无操作流”的最后荣光。整周 `0` 转会，阵容靠天吃饭，结果最后只输了 `23` 分。  

船哥则是标准的两笔小修：`DAY2 Holiday -> Knueppel`，`DAY3 Miller -> Jenkins`。不算华丽，也不算重仓，但胜在踩点够准。  

这场的观感其实很有意思。小火龙整周像是在说“我就不动，你们随意”，结果居然真的硬撑到了最后；船哥则是那种非常典型的务实派，不追花活，只追分。最终双方就差 23 分，属于那种小火龙赛后一定会想一句：“我要是手痒一下，说不定就真翻了。”  

## 弗老大 1461 vs 酸男 1445

这组是本周最像高手过招的一场。  

弗老大两笔操作分别是 `Clifford -> D.Plowden` 和 `Gillespie -> Jenkins`，酸男则是 `文班 -> S.Bey`、`N.Traore -> L.Ball`。双方都不是瞎换，也都不是在赌冷门，属于很标准的“谁更懂本周节奏”。  

最后分差只有 `16` 分。这个级别的对局，一般已经不是谁阵容更强，而是谁少亏一点细节。酸男本周其实已经打得很硬了，但弗老大这边更像一个字：稳。稳着进，稳着排，稳着拿下。  

这种球最气人的地方就在于，你输了，但又很难说自己哪一步是纯送的。只能承认，对手整周比你更像个成熟的人。

## kusuri 1266 vs 老姜 1287

一哥本周开局还是很有一哥风范的，`DAY1 J.Johnson -> P.Banchero`，`DAY1 N.Clifford -> D.Jenkins`，没有多余动作，也没有多余情绪。  

老姜则是另一种路数：`DAY1 Randle -> B.Miller`、`Spencer -> J.Fears`，然后再来一手 `J.Tyson -> N.Alexander-Walker -100`。这种打法属于先在自己腿上来一刀，再看看对面敢不敢和你一起疯。  

最后老姜还真赢了，`1287 : 1266`，只差 `21` 分。  

这场最神奇的点在于：老姜明明吃了 `-100`，最后居然还是赢了。说明什么？说明这周一哥虽然没有崩，但也没打出那种“国一收比赛”的压制感。而老姜则再次证明，范特西里最可怕的不是扣分的人，而是扣完分还真把你干掉的人。

## 马哥 1313 vs 紫葱酱 1354

马哥本周两笔操作都很主流，`J.Johnson -> Mobley`，`Spencer -> Jenkins`，你很难说不合理。  

紫葱酱这边则是 `Westbrook -> Knueppel`、`J.Tyson -> Jenkins`，也是完全围绕本周出勤与热点答案来走。  

两边都没有乱飞，也都没有扣分，最终紫葱酱赢 `41` 分。这个分差属于“看着不大，实际一直追不上”。马哥这周不是没有还手之力，而是每次眼看能接近，紫葱酱总能靠更稳定的排阵把距离再撑回去。  

属于很标准的一场：谁都没出节目效果，但紫葱酱就是更像那个该赢的人。

## 伍家辉 1399 vs 橘队 1203

伍家辉这周赢得很轻松，甚至有点像提前写好答案的那种轻松。`DAY6 Grimes -> T.Jones`，`DAY7 Knueppel -> Jenkins`，两手都很补强，顺着赛程尾声把能吃到的分全吃了。  

橘队这边操作也不算少，`DAY2 Clifford -> D.Carter`，`DAY6 Walker -> P.Reed`，但整队始终没能形成连续冲击。  

最后 `196` 分差，基本就是“从周中开始就没什么悬念”的那种局。伍家辉这周属于稳定运营、稳定收菜；橘队则像是一直在补锅，但锅底早就漏了。

## 大吉鲁 1528 vs 纪导 1411

大吉鲁这一周打出了典型的强队统治力。`DAY4 Podziemski -> Jenkins`，`DAY6 G.Santos -> M.Buzelis`，两笔都很有效，而且整队基础分也够厚。  

纪导这边 `DAY2 文班 -> Jokic`、`Collier -> Knueppel`，其实也不差，甚至是本周很有代表性的两刀。  

问题是，大吉鲁这周不是靠某一两手赢，而是整队都在涨。117 分的差距，说大不算最大，但从比赛体感来说，纪导整周都像在试图咬住一个并不想给机会的人。最终结果也很符合过程：大吉鲁用非常“榜首相”的方式拿下了一场榜首相的胜利。

## AI 1420 vs 柯南 1431

这组是另一场细到不能再细的高手局。  

AI 两笔是 `Podziemski -> Jenkins`、`Filipowski -> Duren`；柯南两笔是 `Collier -> Knueppel`、`J.Miller -> Jenkins`。可以说，两边都在做正确的事，也都没有犯原则性错误。  

最后柯南只赢了 `11` 分。  

这种分差最折磨人，因为输家一定会觉得“我只差一点点”。但范特西最爱干的事就是把“一点点”放大成整个星期的心理阴影。AI 这周已经打得足够像样，奈何柯南在关键节点始终领先半步。最终，这是一场非常柯南式的险胜，也是非常 AI 式的遗憾。

## 桑迪 1385 vs 班班 1447

桑迪本周堪称全联盟最会整活的人。`DAY2` 先正常换两人，`DAY3` 直接开 `AS`，把整队翻了个底朝天：`James`、`Sengun`、`Harden`、`SGA`、`Doncic` 一股脑往里灌，场面一度非常像非法交易市场。  

问题在于，花活的上限虽然高，落袋才叫分。班班这边就显得务实得多：`DAY3 Gillespie -> Jenkins`，`DAY5 Brunson -> Giddey`，两笔够用就行。  

最后班班赢 `62` 分，给这场“艺术足球”般的对决下了一个朴实无华的结论：整活不一定输，但整活的人必须比别人更准。桑迪这次节目效果拉满，比分效果则差点意思。

## Kimi 1496 vs 阿甘 1229

本周最大惨案之一。  

Kimi 两笔操作：`Grimes -> D.Plowden`，`Clifford -> Ball`，没有废手，也没情绪化消费。阿甘则是走到了另一个极端：`G.Santos -> Raynaud`、`N.Queta -> Buzelis`，接着又连续两笔 `-100`，`Grimes -> J.Fears`、`L.Shamet -> Jenkins`，整周一共送掉 `200`。  

最终比分 `1496 : 1229`，差了整整 `267`。  

这场已经不是“换得好不好”，而是“先别自爆行不行”。Kimi 这周属于正常强队发挥，阿甘则属于自己把自己打穿了。200 分的坑在这种赛制里实在太深，后面再怎么努力都像在徒手填海。

## 鬼嗨 1591 vs 凯文 1501

本周最高分大战，鬼嗨笑到最后。  

鬼嗨整周两笔：`N.Clifford -> D.Jenkins`、`J.Gueye -> K.Knueppel`，不花，但都挺毒。凯文则是 `I.Quickley -> D.Carter`、`J.Holiday -> Ball`，同样是实用流。  

如果只看操作，两边都没输。但鬼嗨这周就是那个“每个位置都多冒一点分”的人，最后直接轰到 `1591`，全周最高。凯文 `1501` 放在大多数对局里都够赢了，偏偏碰上本周天花板，只能当背景板。  

这类比赛最没办法，因为你不是打得差，你只是运气不好，正好遇到别人把天花板拆了。

## 本周总结

这一轮联赛最鲜明的气质，就是“热点答案越来越统一，但分差并没有因此变小”。  

大家都在追 Jenkins、Knueppel、Ball，也都知道什么时候该卖人、什么时候该抢出勤，可真正拉开差距的地方，反而变成了两件事：

1. 你有没有在不该扣分的时候上头  
2. 你有没有在最后两天把那一两个位置排到最优

所以本周最具代表性的三种输法也很清楚：

- **文史哲 / AI 这种**：已经打得很好了，但就差最后一口气  
- **尼弟 / 阿甘 这种**：不是没想法，是先把自己打出血了  
- **凯文这种**：分数已经够高，只是碰到了更高的疯子

而最具代表性的赢法则是：

- **堡 / 弗老大 / 大吉鲁**：操作不多，但每一步都像在该动的时候才动  
- **鬼嗨 / Kimi**：不是靠玄学，是靠整周火力线硬压  
- **老姜**：虽然吃了刀，但仍然成功把别人先抬走

GW23 结束，账都结清了。  
有人 1 分过关，有人 200 分埋单，有人高分无福，有人整活翻车。  
范特西最精彩的地方大概就在这：你永远可以在周一觉得自己阵容完美，然后在周日晚上对着一条 `D.Jenkins in` 怀疑人生。
