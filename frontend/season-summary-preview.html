赛季变化曲线

现有 API 能做的

  • 赛季总得分：可以直接从 `history.current` 累加
  • 当前全球排名：能从 `history` / overview 拿到

持有球员情况

这页最适合加“有梗的句子”。因为它不只是统计，而是在讲你和某些球员到底纠缠了多久。

球员羁绊

最长持有`Jalen Johnson` 连续持有 43 天，像是你赛季中段一直舍不得放走的人。
替补常客`Reaves` 在替补席拿到过 7 次双位数，但因为阵型和递补规则没能进有效阵容。
冷门高光你曾经在持有率低于 5% 时押中过一位 50+ 的冷门英雄，这种故事特别值得单独强调。
最专一球员 最常坐板凳的遗憾 深夜神来之笔

现有 API 能做的

  • 如果逐日/逐 event picks 能拉全，最长持有和常替补是可算的
  • “冷门高光”需要当时持有率快照，若官方不保留历史 ownership，就要我们自己存
